## 1.需求:

要求使用纯Python语言编写一个学生管理系统，需要的具体功能是添加每个学生的信息，可以做到删除不要的学生信息，如果有学生的信息有变动需要修改，要达到一个修改学生信息的目的。学校要求对学生信息进行定期的整理查询，做到程序有退出的入口。



## 2.  实现功能:

-  添加学生信息
- 删除学生信息
- 修改学生信息
- 查询学生信息
- 退出



## 3.  环境要求:

- pycharm-professional-2021.2.4
- Windows10或Windows7操作系统
- Python3.9版本
